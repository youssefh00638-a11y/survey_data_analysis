# Survey Data Analysis & Dashboard

##  Project Overview

This project analyzes a real-world survey dataset of data professionals to uncover meaningful insights into the current data job market. It integrates data cleaning, transformation, and interactive visualization to highlight key trends in salaries, job roles, tools, and career satisfaction.

The final deliverable is a Power BI dashboard designed to support exploratory analysis and data-driven decision-making.

---

##  Objectives

- Clean and prepare raw survey data for analysis
- Transform inconsistent and missing data into a structured, usable format
- Analyze key trends:
  - Salary distribution across roles and countries
  - Most commonly used programming languages
  - Job satisfaction and work-life balance
- Develop an interactive dashboard to effectively communicate insights

---

##  Tools & Technologies

- Power BI → Dashboard development and data modeling
- Power Query (M Language) → Data cleaning and transformation
- Excel / CSV → Raw dataset source
- DAX → Calculated measures and KPIs

---

##  Data Transformation Process

The raw dataset required extensive preprocessing before analysis:

### 1. Data Cleaning
- Removed duplicate records
- Handled missing and null values
- Standardized column names
- Removed irrelevant columns (e.g., email, time_taken)
- Cleaned inconsistent text values (e.g., job titles, country names)

### 2. Data Transformation
- Split multi-value columns (e.g., tools and programming languages)
- Created derived columns for improved grouping and analysis
- Normalized categorical variables
- Converted data types (e.g., text to numeric where applicable)

### 3. Data Modeling
- Developed calculated columns and measures using DAX
- Structured the dataset for efficient reporting and performance

##  Dashboard Features

###  Key Insights

- Comprehensive salary analysis:
  - By country
  - By industry
  - By education level
- Career satisfaction across multiple dimensions:
  - Work-life balance
  - Management
  - Career growth (upward mobility)
- Demographic breakdown of survey participants:
  - Age
  - Gender
  - Background
- Key factors data professionals consider when searching for new roles
- Career transition insights:
  - Number of professionals who switched into data
  - Number who started directly in the field

###  Interactive Elements

- Dynamic filters:
  - Country
  - Job Title
  - salary bands
- Fully interactive visuals that respond to user input

---

##  Key Findings

- The dataset includes 620 survey respondents (excluding students)
- The majority of respondents are from:
  - United States
  - India
  - United Kingdom
  - Canada
  - With additional global representation
- 59.5% of respondents transitioned into data careers from other fields
- Python and SQL are the most widely used tools, with R also remaining relevant
- Business Analysts reported the highest satisfaction in:
  - Work-life balance
  - Management
  - Career growth
- Data Scientists and Data Architects reported the highest salary satisfaction
