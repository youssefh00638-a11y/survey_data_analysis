Project Overview

This project analyzes a real-world survey dataset of data professionals to uncover insights about the current data job market. It combines data cleaning, transformation, and interactive visualization to highlight trends in salaries, job roles, tools, and career satisfaction.

The final output is a Power BI dashboard designed to support exploratory analysis and decision-making.

Objectives
Clean and prepare raw survey data for analysis
Transform inconsistent and missing data into a usable format
Analyze trends in:
Salaries across roles and countries
Programming languages used
Job satisfaction and work-life balance
Build an interactive dashboard to communicate insights clearly
🛠️ Tools & Technologies
Power BI → Dashboard creation & data modeling
Power Query (M Language) → Data cleaning & transformation
Excel / CSV → Raw dataset source
DAX → Calculated measures and KPIs
🔄 Data Transformation Process

The raw dataset required extensive cleaning before analysis:

1. Data Cleaning
Removed duplicate records
Handled missing/null values
Standardized column names
Cleaned inconsistent text values (e.g., job titles, countries)
2. Data Transformation
Split columns (e.g., multiple tools/languages)
Created derived columns for better grouping
Normalized categorical data
Converted data types (text → numeric where needed)
3. Data Modeling
Built relationships between tables (if applicable)
Created calculated columns and measures using DAX
Optimized model for performance
📊 Dashboard Features

The Power BI dashboard provides:

🔹 Key Insights
Average salary by job title
Salary distribution across countries
Most popular programming languages
Work-life balance ratings
Career satisfaction levels
🔹 Interactive Elements
Filters (Country, Job Title, Experience)
Drill-down capabilities
Dynamic visuals responding to user input
📈 Key Findings
Data Scientists and Data Engineers tend to have higher average salaries
Python and SQL are the most widely used tools
Work-life balance varies significantly across roles
Higher salaries do not always correlate with higher job satisfaction
